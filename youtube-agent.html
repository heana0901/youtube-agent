⚠️ 브라우저 보안 정책으로 API 호출이 차단됩니다
이 파일을 file://로 직접 열면 브라우저가 외부 API를 차단합니다.
아래 방법 중 하나로 실행해 주세요:
방법 1 — Python (추천)
cd 파일이_있는_폴더
python -m http.server 8080 → 브라우저에서 http://localhost:8080/youtube-agent.html 접속
방법 2 — VS Code Live Server
VS Code에서 파일 열기 → 우클릭 → Open with Live Server
방법 3 — Node.js
npx serve . → 표시된 URL로 접속 후 youtube-agent.html 클릭
📋 사용 방법
